# SeleniumFramework_Udemy Selenium Test Automation Framework
Project Overview

This project is a UI test automation framework for an e-commerce application.
It covers end-to-end user flows such as:

Login functionality
Product selection
Add to Cart
Checkout process

The goal of this project is to demonstrate real-world automation practices using Java and Selenium.

Tech Stack
Java
Selenium WebDriver
TestNG
Maven
Framework Design

The project is built using:

Page Object Model (POM) for better structure and maintainability
Separate layers for tests, pages, and utilities
Reusable methods for actions and validations
Features
Automated login test
Add to cart functionality
Checkout flow validation
Assertions for test validation
Clean and scalable structure

Future Improvements
Add Extent Reports for reporting
Add retry mechanism for failed tests
Integrate CI/CD (Jenkins)
